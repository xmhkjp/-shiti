题目一：使用PHP语言自定义一个函数，此函数作用是将一个句子按单词反序。例如“One World One Dream”，反序后变为 “Dream One World One”。
unction myrev($str){
	$arr = explode(' ',$str);
	$num = count($arr);
	for($i = 0; $i < $num/2; $i++){
		$temp = $arr[$i];
		$arr[$i] = $arr[$num-$i-1];
		$arr[$num-$i-1] = $temp;
	}
	return implode(' ',$arr);
}
$str = 'One World One Dream';
echo myrev($str);
